## Project Overview

English Fun is a browser-based English learning app for Vietnamese children (ages 6+). It features vocabulary flashcards with pronunciation, YouTube song videos, and two interactive games (quiz and spelling).

## Running the App

No build system required. Simply open `index.html` in a browser:
```bash
open index.html
```

## Architecture

### File Structure
- **index.html** (268 lines) - Main page with 4 sections: Home, Vocabulary, Songs, Games
- **app.js** (865 lines) - Application logic: navigation, rendering, audio, quiz, spelling game
- **data.js** (519 lines) - Vocabulary data (17 categories, 170 words), songs list, quiz generation
- **styles.css** (804 lines) - Modern child-friendly design with glassmorphism

### Core Systems

#### 1. State Management (app.js:1-30)
All state lives in module-level variables:
- **Navigation**: `currentSection`, `currentCategory`
- **Scoring**: `totalScore` (persistent via localStorage `englishAppScore`)
- **Streak System**: `streakData` object tracking daily engagement (localStorage `englishAppStreak`)
- **Quiz State**: `quizScore`, `currentQuestionIndex`, `quizQuestions`, `isQuizActive`
- **Spelling State**: `spellingScore`, `currentSpellingIndex`, `spellingQuestions`, `isSpellingActive`, `usedHint`
- **Audio Cache**: `audioCache` Map, `currentAudio` reference

#### 2. Pronunciation System (app.js:390-444)
Two-tier audio system:
1. **Primary**: Free Dictionary API (`https://api.dictionaryapi.dev/api/v2/entries/en/{word}`)
   - Fetches real pronunciation audio
   - Results cached in `audioCache` Map
   - Preloads audio when category changes (app.js:47-58)
2. **Fallback**: Web Speech API (`speechSynthesis`)
   - Used if API fails or audio unavailable
   - Configured for kids: slower rate (0.8), higher pitch (1.1)

#### 3. Streak System (app.js:88-180)
Gamification to encourage daily practice:
- Tracks consecutive days of activity
- Persists to localStorage (`englishAppStreak`)
- `recordActivity()` called on quiz/spelling completion
- Displays fire animation for streaks Ōēź7 days
- Automatically resets if user misses a day

#### 4. Quiz Game (app.js:497-657)
Multiple-choice quiz system:
- 5 questions per quiz, 4 options each
- Questions generated by `generateQuizQuestions()` in data.js:478-519
- +10 points per correct answer
- Category selection via `.quiz-category-btn` buttons
- Flow: Start ŌåÆ Questions ŌåÆ Result ŌåÆ Restart

#### 5. Spelling Game (app.js:683-865)
Type-the-word game:
- 5 words per game from selected category
- Audio pronunciation button (­¤öŖ)
- Vietnamese hint button (­¤ÆĪ) - reduces score if used
- +15 points per correct word (no partial credit)

#### 6. Songs/Videos (app.js:447-494, data.js:426-475)
YouTube embed system:

- Modal-based video player
- Songs rendered as cards with emoji thumbnails

### Adding New Content

**New Vocabulary Category**:
1. Add category object to `vocabularyData` in data.js
2. Add category button to **both** `.category-selector` sections in index.html:
   - Line ~84-105 (Vocabulary section)
   - Line ~131-142 (Games section)
3. Use `data-category="categoryKey"` attribute

**New Song**:
Add to `songsData` array in data.js:426-475 with YouTube video ID.
